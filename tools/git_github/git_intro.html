Git Introduction

Git is open source distributed version control system.


Features of Git -

1. Open Source
2. Scalable
3. Distributed
4. Security
5. Speed
6. Supports non-linear development
7. Branching and Merging
8. Data Assurance
9. Staging Area
10. Maintain the clean history


Benefits of Git -
 
1. Saves Time
2. Offline Working
3. Undo Mistakes
4. Track the changes
