Docker Command Cheat Sheet


1. To see all the images present in your local machine

docker images

2. To find out images in docker hub

docker search jenkins

3. To download image from docker hub to local machine

docker pull jenkins

4. To start/run the container in interactive mode and give it a name

docker run -it --name somename ubuntu /bin/bash

5. To check service is started or not

service docker status or systemctl start docker

6. To start container

docker container start container_name

7. To go inside container

docker container attach container_name or docker attach container_name

8. To see all the container

docker container ls -a or docker container ps -a or docker ps -a

9. To see all running container

docker container ps or docker container ls or docker ps

10. To stop container

docker stop container_name

11. To delete container_name

docker rm container_name

12. To get the information about the docker

docker info
